### &#x266f; Remove Cache File 
```
1) latexmk -c: remove certain file specified in .latexmkrc 
2) latexmk -C: remove all files generated by ":VimtexCompile<CR>", including .pdf

3) Mapping <leader>c for ":VimtexCompile<CR>"
           ---------
```
